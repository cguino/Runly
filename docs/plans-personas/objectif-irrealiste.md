# Persona « objectif-irrealiste »

- **Objectif** : semi le 2026-08-30, ambition chrono (1h20)
- **Contexte** : 3 séances/sem (mar, jeu, dim), volume 20 km/sem, antécédent < 12 mois : non
- **VMA** : 14,0 km/h

**Résultat : objectif jugé irréaliste** (`too_few_weeks`, `volume_gap`, `pace_above_capacity`).

Alternatives proposées :
- passer l’ambition à « finir »
- viser une date ultérieure (≥ 2026-10-05)
- viser un autre objectif : 10k
